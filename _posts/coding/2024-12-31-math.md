# 행렬
단위행렬 = [[1,0,0],[0,1,0],[0,0,1]]
전치행렬 = [[10,20],[30,40],[50,60]] => [[10,30,50],[20,40,60]]

## 부울대수

최소항 - 곱
최대항 - 합

카르노맵 - 짝수개로 묶어서 최소항, 최대항 계산
AND,OR,NOT,XOR

NAND => AND에 NOT
NOR => OR예 NOT

## 정사각행렬의 행렬식
|A| = [[11,12],[21,22]] = 11*22-12*21
|B| = [[11,12,13],[21,22,23],[31,32,33]] = (11*22*33+12*23*31+13*32*21) - (13*22*31+23*32*11+33*21*12)
