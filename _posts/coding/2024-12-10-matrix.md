# 행렬   

2차원배열[[],[]] - 1차원배열이자 2차원배열(a[0][0])
3차원배열[[[],[]],[[],[]]] - 1차원배열이자 2차원배열이자 3차원배열(a[0][0][0])

- row(행),column(열),cell로 구성
- 행렬의 곱 -> 행열 곱 행X열 -> ex> 2X3 곱 3X2 => 3으로 같아서 곱셈가능/ 결과는 2행2열
