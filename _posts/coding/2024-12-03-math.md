## 시그마 (합)
````javascript
// 시그마(Summation)
let sum = 0;
for(let i=0;i<=5;i++){
	sum+=i;
}
for(let i=6;i<=10;i++){
	sum+=i;
}
document.write(sum);
````

## 프로덕트 (곱)

````javascript
//파이(Product)
let pro = 1;
for(let i=1;i<=5;i++){
        pro*=i;
}
document.write(pro);
````
